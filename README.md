# Developer Burnout Decision Tree

## Project Goal

This project is a full-stack web application that predicts a developer's burnout level using a manually implemented Decision Tree algorithm. The application also visualizes the generated decision tree using an interactive graph.

## Algorithm

The project implements a Decision Tree algorithm manually using **Entropy** and **Information Gain** as the splitting criterion.

No machine learning libraries (such as Scikit-learn) were used.

Entropy and Information Gain were chosen because they provide a clear way to measure how well each feature separates the data and are commonly used in the ID3 Decision Tree algorithm.

## Dataset

The dataset contains the following features:

- Sleep
- Meetings
- Workload
- Overtime
- Stress
- Remote
- Age
- ExperienceYears
- Gender
- Role
- Exercise

Target values:

- Healthy
- Risk of burnout
- Vacation required
- Critical condition
